# Automatic Power Cut System

# Overview
The **Automatic Power Cut System** is an Arduino Uno-based embedded systems project that protects electrical appliances from overvoltage conditions. It continuously monitors the input voltage and automatically disconnects the power supply using a relay when the voltage exceeds a safe limit. Once the voltage returns to the normal range, the system restores the power automatically.

This project demonstrates real-time voltage monitoring, relay control, and automation using embedded systems.

---

# Features
- Automatic overvoltage detection
- Real-time voltage monitoring
- Automatic power cut and restoration
- Relay-based load protection
- LED status indication
- Simple, low-cost, and reliable design

# Working
The Arduino continuously reads the input voltage from the potentiometer. If the voltage exceeds the preset threshold, it activates the relay to disconnect the load and protect connected devices. When the voltage returns to a safe level, the relay reconnects the power automatically.
